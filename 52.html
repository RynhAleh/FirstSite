Урок 52. Селекторы, Урок 53. Группировка селекторов. Домашнее задание:

Упростите код, используя группировку селекторов:
h1 {
   text-align: center;
}
h2 {
   text-align: center;
   color: blue;
}
h3 {
   text-align: center;
   font-size: 16px;
   color: red;
}
}

Решение:

h1, h2, h3 {
   text-align: center;
}
h2 {
   color: blue;
}
h3 {
   font-size: 16px;
   color: red;
}
