Урок 51. Форматирование текста. Часть 2.Домашнее задание:

Сократите код, используя свойство-сокращение font:
p {
   font-family: "Times New Roman";
   font-size: 13px;
   line-height: 20px;
}
 Сократите код, используя свойство-сокращение font:
p {
   width: 300px;
   font-family: Arial;
   color: red;
   font-size: 40px;
   font-weight: bold;
}
 Сократите код, используя свойство-сокращение font:
p {
   font-family: Arial;
   font-size: 40px;
   font-weight: bold;
   text-indent: 50px;
   font-style: italic;
   line-height: 60px;
}

Решение:

p {
   font: 13px/20px "Times New Roman";
}

p {
   width: 300px;
   color: red;
   font: bold 40px Arial
}

p {
   text-indent: 50px;
   font: italic bold 40px/60px Arial
}


Решение исходило из того, что свойство-сокращение font может включать в себя следующие 6 свойств:
font-style normal; italic; oblique; inherit; initial; unset
font-variant small-caps; common-ligatures small-caps; inherit; initial; revert; revert-layer; unset
font-weight normal; bold; lighter; bolder; 200; 900 и т.д.
font-size / line-height 10px/1.5; 30px/20px и т.д.
font-family 'Times New Roman','Arial'; serif; sans-serif;monospace;fantasy и т.д.

Порядок расположения на примере:
body { font: italic small-caps bold 16px/1.5 Arial, sans-serif; }
